Building (win32 only)
==================
1. Open a command prompt window.
2. Switch to the MK2 dir.
3. Execute `mountdrive.bat`
4. Switch to the W: drive. (it default mounts it to W)
5. Then run tools\_missile.bat
6. In the new command prompt run build to build the OS
